=head1 NAME

NCM::ofed - NCM ofed configuration component

=head1 SYNOPSIS

=over

=item Configure()

This component configures OFED.

=item  /software/components/ofed/openib

=item config 
Config file loaction for openib (default /etc/infiniband/openib.conf).

=item options
Config options

=item modules
Modules to load

=item hardware
Hardware modules to load

=back

=head1 RESOURCES

=over

=item /software/components/ofed/active : boolean

activates/deactivates the component.

=item /software/components/ofed/servers/ : list of time servers

=item /software/components/ofed/clientnetworks : list of { net=, mask= }

optional: clients that can use this server to synchronize. Default allows
connections from localhost only.

=back

=head2 Components to be run after:
